# PatiOS

My own WebOS! It's pretty rudimentary right now, but I think it's pretty cool. I had lots of fun making it.

## Features
- Windows
  - Opening and closing
- Apps and desktop icons
  - Welcome App with information about me
  - "System Stats" app - just a cool little animation I made with some binary
  - Calculator app - fully functional
- Top bar with time and minimized apps
- Background image that I really like
- Some smooth styling IMO

## Take a look
[Live Demo](https://pati08.github.io/WebOS/)

## AI Usage
AI was used only through LLM autocomplete in Zed and the styling of the calculator app
